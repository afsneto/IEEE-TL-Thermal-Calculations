# IEEE 738-2006
Thermal Rating Graph of Bare Overhead Conductors following the IEEE 738-2006.

## Input data
1. Solar heating data
- Specified global solar radiation (W/m²)
- Day of year
- Line direction (°)
- Latitude (°)
- Sun time
- Atmosphere

2. Weather data
- Air temperature (°C)
- Wind speed (m/s)
- Wind to conductor angle (°)

3. Conductor data
- Cable file name
- Elevation (m)
- Absorptivity
- Emissivity

4. Calculation data
- Minimum temperature
- Maximum temperature
